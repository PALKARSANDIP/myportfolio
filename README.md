# Sandip Palkar — Portfolio

Built with **Next.js 14** + **Tailwind CSS** (JSX version)

## 🗂 Folder Structure

```
portfolio/
├── app/
│   ├── globals.css
│   ├── layout.jsx
│   └── page.jsx
├── components/
│   ├── ThemeProvider.jsx
│   ├── Navbar.jsx
│   ├── Hero.jsx
│   ├── About.jsx
│   ├── Skills.jsx
│   ├── Experience.jsx
│   ├── Projects.jsx
│   ├── Contact.jsx
│   └── Footer.jsx
├── public/
│   ├── favicon_boy.svg       ← favicon
│   ├── avatar_curly.svg      ← avatar (fallback photo)
│   └── Image/
│       └── SandipPalkar_Portfolio.jpg  ← add your real photo here
├── package.json
├── tailwind.config.js
├── next.config.js
└── .gitignore
```

## 🚀 Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 🌐 Deploy to Vercel

1. Push to GitHub
2. Go to vercel.com → New Project → Import repo
3. Click Deploy ✅
